Installation help supplement

Remember you need to have java installed to run this program : you can start here.

What is good here is that you don't have to unjar, unzip, untar or what ever. (If you understood that, you are nearly done!)
Just click on the following : Sourceforge
.

Alternatively you can download the big jar file in a folder, go in the latest and run : java -jar pms-0.x.x.jar or java -jar <what ever you renamed it>

If it doesn't work!

Yes, a successful software installation is sometimes the tricky bit as there can be OS specificities on the way.
So if you have a problem installing, and you are one of these who don't give up easily, please follow the instructions below.

Have you tried this link : Sourceforge ?

If this still doesn't work you can download the jar from here (click on the latest release and then the jar file) : http://sourceforge.net/project/showfiles.php?group_id=214811
Don't unpack it. But save it somewhere on your file system. When it's all here, just go in the folder and run java -jar pms-0.x.x.jar on the command line.
On MS Windows you should actually be able to run it just by double clicking on the jar.

You should then just have to follow the instructions step by step.

Good luck.
